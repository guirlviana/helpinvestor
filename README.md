<h2 id="started">‚è≤Ô∏è Getting started</h2>

The app are running with `Python 3.10.12`

- Clone the project
- Create a virtualenv

```bash
python3 -m venv venv
```
- Activate
  
```bash
source venv/bin/activate
```

- Install libraries and dependencies

```bash
pip install -r requirements.txt
```
